# sqlite-CRUD-CP
This project is a Fragment-based Android app using Java and SQLite to perform full CRUD operations. It features a Sign Up screen to create users, a Login screen for authentication, and a Manage Account section to Update (Forgot Password) or Delete records, all handled within a single Activity for a smooth UI flow.

# SQLite CRUD Fragment Application
A simple Android application built with Java and SQLite implementing full CRUD operations using Fragments.

## Features
- **Sign Up (Create):** Register users with Username, Email, Password, and DoB.
- **Login (Read):** Verify user credentials from SQLite.
- **Forgot Password (Update):** Update password using the registered email.
- **Delete Account (Delete):** Remove user data from the database.

## Technologies Used
- Java
- Android XML Layouts
- SQLite Database
- Android Fragments
